# 🚀 QUICK START - Grammar HUB en GitHub Pages

## En 3 Pasos

### 1️⃣ Crear repo en GitHub
```
Ve a https://github.com/new
→ Nombre: Grammar-HUB
→ Public
→ Create Repository
```

### 2️⃣ Conectar y subir
```bash
cd "c:\Users\moral\Desktop\Apps\Grammar HUB"

git remote add origin https://github.com/TU_USUARIO/Grammar-HUB.git
git push -u origin main
```

### 3️⃣ Activar GitHub Pages
En tu repo:
- **Settings** → **Pages**
- Source: Branch `main`, folder `/ (root)`
- Click **Save**

**Tu app estará en:**
```
https://tu_usuario.github.io/Grammar-HUB/
```
